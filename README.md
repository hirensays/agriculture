# Agriculture_Analysis
Objective of the mini project is to analyze the various states crop yield data and to see the cropping pattern among the farmers.Some of the tasks implemented in the project is highlighted below:
1.  Economical analysis of the overall farming sector in India.
1.  Analysis of crop selection pattern among farmers based on yield and price.
2.  Effects of temperature, rainfall on crop yield.
3.  Implementation of the crop recommendation algorithm based on weather condition, yield/area and selling price.
4.  State Wise analysis of the various crops and its production.
